Test Strategy Overview

Detect

Validate hardware data capture accuracy, environmental resilience, device calibration, and failover. Covers WIM (±2%), ANPR (≥95% day / ≥90% night), RFID (≥99% at 10m), AVC (±5%), Axle Counter (100%), Laser Curtain, and signal synchronization using HiL and physical simulators.

Fuse

Verify sensor fusion, RFID/ANPR correlation with confidence scores, mandatory field completeness, orphan data prevention, store‑and‑forward edge integrity, duplicate prevention, and decision tree logic under peak‑hour concurrency (50 TPM).

Validate

Business logic checks against Masar/Tahseel, configuration sync, RBAC enforcement, NOC/permit grace periods, classification accuracy, and API integration with mocked back‑end services.

Action

Financial deduction precision, wallet insufficient‑funds handling, fine issuance compliance, notification delivery (SMS/Email), and end‑to‑end latency measurement (<2s normal, <10s under load).
